暴力

朴素

\(O(n^2)\) 枚举所有子串然后 \(O(n)\) 判断是否回文，总复杂度 \(O(n^3)\)

小优化

回文串长度为奇数，对称轴就在中间字符上。 长度为偶数，对称轴就在靠中间的两个字符之间。 可以从 1 ~ n 枚举对称轴然后往外暴力扩展，复杂度为 \(O(n^2)\)

\(Code\) : 

int work(string s) {
    int ans = 0;
    for(int i = 1, pos;i < s.size(); ++i) {
        pos = i + 1;
        while(s[pos] == s[2 * i - pos] && 2 * i - pos >= 0 && pos <= s.size() - 1) ++pos;
        ans = max(ans, pos - i);
        
        pos = i + 1;//对称轴为 s[i] 后面的空隙
        while(s[pos] == s[2 * i - pos + 1] && 2 * i - pos >= 0 && pos <= s.size() - 1) ++pos;
        ans = max(ans, pos - i);
    }
    return ans;
}
这种暴力即使经过优化也有很大弊端： 1. 要考虑回文长度奇偶性 2. 没有利用到前面查找的结果

对于第二条，在 \(manacher\) 算法中，当（前面组成）与（当前要扩展的部分）成分相同的时候直接拿前面部分的扩展结果来作为初始状态使用，大大减小了其复杂度，可以降低到 \(O(n)\) 。

马拉车

解决奇偶性

对于第一条，像 aba 这样回文长度为奇数的串，对称轴落在字母上；像 baab 这样的回文长度为偶数的，对称轴就落在了两个字母之间，为了进行统一，我们用一些未出现的字符进当成隔板插在字符串中，这样，无论是 ~|a|b|a| 还是 ~|b|a|a|b| ，它们的对称中心都会落在新字符串的字母上。可以看到我放了一个 ~ ，其实就是为了使其扩展到边界时无法继续向外扩展。

前文也说了该算法的机制，可他具体又是怎么运行的呢？ - 假如有一个回文串 S ，它的对称轴左边有一个区间 A，那么对称轴右边一定有一个区间 B 是关于 A 对称。 比如 abcxcba 中，左侧有一个 abc ，右侧就有一个对称的 cba。 - 如果这个区间 A 本身是回文串，显然区间 B 也是一个回文串， 而且在他们所处的回文串 S 中半径相等。 - 如果我们要确定以下标 i 为中心的回文半径，我们可以找到属于它的区间 A，A的回文半径肯定小于等于以 i 为中心的回文半径。

变量约定

  • 使用一个 p 数组记录每一个位置的最大回文半径
  • r 来记录前面找到的回文串中最靠右的右边界的位置
  • mid 记录与其对应的回文中心。

当对 i 进行扩展时，有两种情况： - \(r < i\) 需要从 0 开始扩展 - \(r \geq i\) 时，需要找到 mid 左侧的区间 A ，把 A 的回文半径作为 i 的初始回文半径继续扩展 设 i 关于 mid 对称在区间 A 的回文中心是 j ，根据中点公式 \(\frac{i + j}{ 2 }= mid\) 可得 \(j = mid * 2 - i\)

#include <bits/stdc++.h>
#define N 11000010
using namespace std;

int n, p[N << 1], ans = 1;
char a[N], s[N << 1];

void manacher() {
    s[0] = s[1] = '#';
    for(int i = 0; i < n; i++) {
        s[i * 2 + 2] = a[i];
        s[i * 2 + 3] = '#';
    }
    n = n * 2 + 2;
    s[n] = 0;

    int r = 0, mid;
    for(int i = 1; i < n; i++) {
        if(i < r)                                 // 可以白嫖
            p[i] = min(p[(mid << 1) - i], r - i); // 左侧区间A可能有超过S的部分，所以初始状态最大就是r - i
        else
            p[i] = 1;
        for(; s[i + p[i]] == s[i - p[i]]; ++p[i]) // 暴力扩展
            if(p[i] + i > r) {                    // 更新r与mid
                r = p[i] + i;
                mid = i;
            }
    }
}

int main() {
    scanf("%s", a);
    n = strlen(a);
    manacher();
    for(int i = 0; i < n; i++)
        ans = max(ans, p[i]);//遍历找最大值
    printf("%d", ans - 1);
    return 0;
}
